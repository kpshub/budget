  2. Search App registrations → click New registration
  3. Name it "Budget Dashboard" · Supported account types: Personal Microsoft accounts only
  4. Redirect URI → Select Single-page application (SPA) → paste this URL:
  5. Click Register
  6. Go to API permissions → Add a permission → Microsoft Graph → Delegated → add Files.Read and User.Read
  7. Copy the Application (client) ID shown on the Overview page
Once hosted on GitHub Pages or Netlify, update the Redirect URI in Azure to your hosted URL — then both you and your wife can access it from any device.
